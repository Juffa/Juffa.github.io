Frens

Okay, so a "Fren" is supposedly some swedish meme, and a swedish friend of mine made a base version and asked if I want to make custom versions of them...
So these are what came out of that.
They were supposed to be very low effort, with no real hands or feet, but then I went overboard with them.
Each of them is based on a player's avatar on ChilloutVR.
Someone made a nav mesh follower mod for the game, and these can work with that, so they can run after you or do some tasks you can assing to them, like fetch an item or so...
There is going to be more of these...

Frens

Joo eli siis "Fren" on ilmeisesti jokin ruotsalainen meemi, ja eräs ruotsalainen kaveri teki siitä perusversion ja kysyi haluanko tehdä kustomoituja versioita...
Nämä ovat siitä syntyneitä.
Niiden oli tarkoitus olla todella vähällä vaivannäöllä tehtyä mutta meni vähän överiksi.
Jokainen näistä perustuu pelaajan avatariin ChilloutVR:ssä
Joku teki nav mesh follower modin kyseiseen peliin jonka avulla muunmuassa näitä hahmoja voi käyttää followereina, joita voi komentaa tekemään kaikenlaisia asioita.
Näitä olisi tarkoitus tehdä lisää...
